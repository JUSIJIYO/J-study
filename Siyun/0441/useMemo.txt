useMemo
: 연산 결과 캐싱해서 불필요한 계산 줄이는 Hook
- 사용 이유
: 성능 최적화
 이미 계산한 값인데 또 다시 계산하는 걸 막으려고 (불필요한 작업 줄임)

 import { useMemo } from "react";
function App({ num }) {
  const result = useMemo(() => {
    console.log("계산 실행");
    return num * 2;
  }, [num]); -> 두번째 인자 = 의존성 배열
  return <div>{result}</div>;
}
-> 처음 렌더링 (계산 실행) -> 의존성 배열 값 바뀌면 다시 계산 -> 값 안 바뀌면 기존 값 재사용
